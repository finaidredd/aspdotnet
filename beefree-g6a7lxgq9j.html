During this tech talk we will establish a base knowledge of .NET Core technologies and how they compare to similar tech such as the .NET Framework and Mono.

 

We will also grok the mysteries of "just what is .NET Standard and attempt to demystify the verbiage in play.

 

Once the fundamentals are established, we will explore the tools and wiring for building ASP.NET Core applications using technologies such as WebAPI and MVC.

 

A big part of .NET Core is the ability to run cross platform, so expect to see some demos on MacOS as well as Linux .

 

What about IDE if you’re targeting Linux? No problem, we’ll use VS Code as well as portions of the show and tell.

 

Expect to have a fundamental knowledge of .NET Core, .NET Standard, Visual Studio Code, and Visual Studio for Mac, as well as how to work with the tools in the various operating systems.
